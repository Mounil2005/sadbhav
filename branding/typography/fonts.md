# Sadbhav Hospital — Typography

## Typefaces

### Display / Headings
**Playfair Display** (Google Fonts)  
- Weight: 600 (SemiBold), 700 (Bold)  
- Usage: Section headings, hero title, doctor name, article titles  
- Tailwind class: `font-display`

### Body / UI
**Inter** (Google Fonts)  
- Weight: 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)  
- Usage: Body text, labels, nav links, buttons, captions  
- Tailwind class: `font-sans` / `font-body`

## Type Scale

| Class | Size | Usage |
|---|---|---|
| Hero H1 | 64px / 4rem | Main hero headline |
| Section heading | 36–48px | Section H2s |
| Card heading | 18–24px | Card/article titles |
| Body large | 18–20px | Lead paragraphs |
| Body | 14–16px | Standard body text |
| Label | 10–12px | Section labels, uppercase tags |
| Caption | 10–11px | Meta, subtext |

## Loaded via
`index.html` Google Fonts preconnect:
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Playfair+Display:wght@600;700&display=swap" rel="stylesheet">
```
