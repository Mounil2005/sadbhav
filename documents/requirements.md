## Project Scope

A production-ready hospital website for Sadbhav Hospital — a pulmonology and critical care practice based in Jamnagar, Gujarat.

### Completed
- [x] Single-page React + Vite + Tailwind CSS frontend
- [x] 10 sections: Hero, Highlights, About, Services, Doctor, Facilities, Blog, Testimonials, Contact, Footer
- [x] Cal.com appointment booking integration (floating button + modal trigger)
- [x] Google Maps embed (live hospital location)
- [x] WhatsApp contact link
- [x] Responsive across mobile, tablet, desktop
- [x] Scroll reveal animations
- [x] Accessibility: skip nav, ARIA labels, keyboard focus, semantic HTML
- [x] Real assets: logo, doctor photo, ICU photo, reception photo
